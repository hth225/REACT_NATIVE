# REACT NATIVE Weather App
## What is this Project?
- 현재 위치를 위도, 경도값으로 얻어내어 openwathermap api에 넘겨준다
- 넘겨준 값을 토대로 해당 위치의 날씨를 json 형태로 받는다
- 넘겨받은 값을 해당하는 아이콘과 텍스트로 날씨를 표시하여 준다
